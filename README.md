# Washington Civic Clarity Dashboard

This project is a prototype civic transparency tool designed to help people understand Washington State legislation in plain language.

Government bills are often written in complex legal language. The goal of this dashboard is to translate those documents into clear operational meaning so citizens can quickly understand what a bill actually does.

## Core Feature

The Legislative Language Analyzer interprets bill text and produces a simple explanation of:

- what changed
- who it affects
- what action is required
- the real-world operational effect

Instead of reading dense legal language, users see a structured summary of the bill’s impact.

## Project Goal

Build a semantic interpretation layer that sits between legal documents and the public.

The system extracts key signals from legislative language (requirements, permissions, restrictions, deadlines) and converts them into structured explanations.

## Current Status

Prototype.

The current version demonstrates:

- example bill cards
- structured impact summaries
- links to official Washington Legislature resources

Future versions will automate bill analysis and expand the dashboard.

## Why This Matters

Many citizens want to understand legislation but do not have the time or legal background to parse complex bill text.

This project explores whether semantic analysis can make legislation easier to understand and navigate.
