Washington Civic Dashboard

Translate Washington bill records into clean, usable structure, then expose the rule logic in a low-load interface.

1. Search the local bill catalog by bill number, title word, or known variation.

2. Load a selected bill record into the translator.

3. Review who it applies to, required actions, allowed actions, prohibited actions, timing, and source anchors.

4. Use the official links to verify the source record.

Architecture

A simple view of the dashboard flow. Each layer stays focused so people can understand the system without reading a dense diagram.
